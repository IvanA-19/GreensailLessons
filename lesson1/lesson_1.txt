1 Создание простейшего эхо-бота на библиотеке pytelegrambotapi

1.1 Установка библиотеки
Для установки библиотеки пишем в терминал pip install pytelegrambotapi

1.2 После завершения установки создаем в корневой папке проекта несколько основных файлов:
bot.py
config.py

1.3 В файле config.py размещаем вспомогательные данные
1.4 В файле bot.py располагаем основной код бота

Пример кода будет приведен в папке echo_bot

2. В папке bot_keyboards можно найти информацию по написанию клавиатур для бота

3. В папке home_work_1 будет код с разбором домашнего задания по занятию 1
